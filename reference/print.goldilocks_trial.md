# Print an adaptive trial trace result

Prints the final trial summary and reports how many interim looks were
completed when survival_adapt returns a goldilocks_trial object.

## Usage

``` r
# S3 method for class 'goldilocks_trial'
print(x, ...)
```

## Arguments

- x:

  A goldilocks_trial object.

- ...:

  Additional arguments passed to print.data.frame.

## Value

The input object, invisibly.
